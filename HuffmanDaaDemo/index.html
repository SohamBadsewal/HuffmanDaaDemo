Huffman Coding

Lossless data compression via optimal prefix-free binary trees.

15 characters × 8 bits = 120 bits (unencoded)

Character frequencies: A=5, B=1, C=6, D=3

ALGORITHM

Construction Steps

  • 01

    Create a leaf node for each character. The leaf node contains the frequency of the character.

  • 02

    Arrange all nodes in increasing order of their frequency value. These are maintained in a priority queue Q.

    For this example: Q = [B:1, D:3, A:5, C:6]

  • 03

    Extract the two minimum frequency nodes. Create a new internal node with frequency = sum of the two. Make them left and right children. Insert new node back into Q. Repeat until Q has one node.

  • 04

    The remaining single node is the root. The tree is the Huffman Tree.
